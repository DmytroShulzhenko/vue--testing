# Первый проект

## Структура проекта

- `scss/` - папка ...
- `js/` - папка ..


## Установка

Клонируем репозиторий
```
git clone git@github.com:DmytroShulzhenko/vue--testing.git
```

Переходим в папку проекта
```
cd vue--testing
```

Устанавливаем зависимости
```
npm install || npm i
```

Запускаем проект
```
npm start || npm run start
```
